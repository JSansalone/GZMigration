br.com.gz.migration.datafile

Class DataFile

public abstract DataFile

Representa um arquivo de dados e permite manipular as informações contidas no arquivo físico como: retornar os registros, percorrer os registros, contar a quantidade de registros, etc. As subclasses são responsáveis por implementar opcionalmente o singleton para garantir a segurança do acesso ao arquivo de dados.

checkHeaderPolicy

public abstract boolean  checkHeaderPolicy()

Método para verificar os nomes das colunas.

Responsabilidades:
1- Verificar se todas as colunas estão presentes
2- Verificar se os nomes das colunas estão na ordem correta
3- Verificar se o tipo de dado das colunas está correto
4- Lançar se o tipo de dado da coluna for inválido (não for String)
5- Lançar se alguma coluna obrigatória não for encontrada
